📖 安装指南

Agent花园·Code 有两种安装方式，选一个适合你的。

🔑 准备工作：注册你的 DeepSeek API Key（自备 Key）

Agent花园·Code 使用的是你自己的 DeepSeek 账号，数据只经过你自己的 Key，安全放心。你也不需要花钱——DeepSeek 注册就送免费额度，足够用很久。

1 打开 DeepSeek 官网

访问 platform.deepseek.com，点击"注册"。

2 创建 API Key

登录后，左侧菜单找到 API Keys → 点击 创建 API Key → 复制生成的 Key（一串以 sk- 开头的字符）。

3 把 Key 交给我们（或自己配置）

复制好 Key 后，往下看安装方式👇

💡 为什么要自备 Key？ 你自己的 Key → 数据走你自己的账号 → 隐私不外泄。
而且 DeepSeek 注册赠送免费额度，绝大多数个人用户几个月都用不完，不用额外花钱。

🚀 方式一：远程安装（推荐）

把 DeepSeek Key 发给我们，我们远程帮你装好。全程你可以在屏幕上看着我们操作，安全放心。

准备工作

  • ✅ 已注册 DeepSeek 并创建了 API Key
  • ✅ 一台能上网的电脑（Windows 10/11 或 macOS 12+）
  • ✅ 已加入知识星球（基础版 ¥199 或 会员版 ¥365/年）

安装流程

1 下载向日葵客户端

点击下方按钮，从向日葵官网下载客户端（安全无毒，放心下载）。

⬇️ 下载向日葵客户端

官网链接，安全可靠 · 文件很小，下载只需几秒

2 运行向日葵，获取连接码

双击运行下载的向日葵程序，无需安装。打开后，你会看到本机识别码和临时密码。把这两个数字和你的 DeepSeek API Key 一起发给我们。

3 我们远程安装

我们远程连接你的电脑，帮你把 Key 配置好，装好 Agent花园·Code。

整个操作你都能在屏幕上看到——装完之后断开连接，关闭向日葵即可，不留任何后台程序。

💡 关于向日葵的顾虑： 如果担心"对方发来的不是木马吧？"——没关系，你直接从向日葵官网下载，不存在安全问题。 你还可以选择自己先下载研究一下，什么时候准备好了联系我们就行。

⚡ 方式二：自己装 + 一键配置脚本

你有 Key，想自己装，没问题。我们提供一个配置脚本，运行后自动把你的 Key 嵌入到系统里，不用手动改文件。

1 下载安装包

在产品页选择你的版本，下载对应系统的安装包。

2 运行安装

Windows：右键点击 install.bat → "以管理员身份运行"

Mac：终端里运行 chmod +x install.sh && ./install.sh

3 输入你的 DeepSeek API Key

安装过程中，脚本会提示你输入 DeepSeek API Key。把刚才复制的 Key 粘贴进去，按回车。脚本自动配置好，无需手动操作。

4 安装完成，直接使用

全程大约 5-15 分钟，完成后打开 Agent花园·Code 就能用了。

💡 自己装的好处： Key 从开始到结束都在你自己的电脑上，不用传给任何人，彻底放心。

💻 系统要求

  • Windows：Windows 10/11（64位），8GB 内存，10GB 磁盘空间，管理员权限
  • Mac：macOS 12+，Apple Silicon 或 Intel，8GB 内存，10GB 磁盘空间
