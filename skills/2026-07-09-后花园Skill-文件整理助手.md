## 这个 Skill 是干什么的

当你跟 Claude Code 说"帮我整理一下桌面"，它会自动分析你的文件：按类型分类（文档、图片、压缩包、安装包）、标记重复文件、建议更清晰的目录结构、清理临时文件和垃圾。

**核心价值：** 不需要你自己想"这个文件放哪"——Claude 帮你想，整理完告诉你它做了什么。

## 什么时候用

- 你的下载文件夹乱得无从下手
- 找不到不久前的文件到底存哪了
- 硬盘空间不知不觉就满了
- 项目文件散落在桌面、文档、下载三个地方
- 想批量重命名但不想一个个点

## 怎么用

装了这个 Skill 后，直接在 Claude Code 里说：

> "整理一下我的下载文件夹"

> "帮我把桌面按项目归类"

> "找出所有重复的 PDF 文件"

> "把上周的图片移动到 图片/2026-07/ 文件夹"
