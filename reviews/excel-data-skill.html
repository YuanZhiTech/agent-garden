评测对象

document-skills/xlsx — 让 Claude Code 具备专业的 Excel/电子表格处理能力。支持创建、编辑、公式计算、数据分析、可视化。

为什么选这个测

元智的市场调研数据：Excel 数据清洗 Skill 在腾讯 SkillHub 上 98 万+ 下载量，评分 4.7。这是验证过的刚需——不是我们猜的，是市场告诉我们的。

怎么装

# 从矿脉②克隆（一次克隆，永久使用）
git clone https://github.com/ComposioHQ/awesome-claude-skills.git

# 在 Claude Code 中加载
claude --skill-dir ./awesome-claude-skills/document-skills/xlsx

跑起来怎么样

装好后，你让 Claude 做的事：

  • 查询数据： "帮我查一下这个表格里 2025 年的总销售额是多少"
  • 创建表格： "帮我创建一个周报模板，带自动求和、条件格式、折线图"
  • 数据清洗： "这个表格里日期格式不统一，帮我标准化成 YYYY-MM-DD"
  • 公式计算： "在 F 列加上一个公式，计算毛利率 = (收入-成本)/收入"

都能真实生成 .xlsx 文件，下载到本地就能用。

真实效果

  • 简单任务（查个数据/简单公式）：秒回，几乎不出错
  • 中等任务（多表关联/带格式的报表）：质量高，但需要你描述清楚需求
  • 复杂任务（带 VBA 宏/数据透视表）：能处理，但建议生成后人工复核

踩坑

  1. 中文列名可能有问题 — Excel 公式对中文列名的支持不如英文稳定，建议列名用英文，数据用中文
  2. 格式偶尔会丢 — 生成的 xlsx 打开后，某些单元格格式可能和预期的有细微差异
  3. 大文件性能 — 超过 1 万行的表格处理速度会明显变慢

适合谁

  • 周报/月报要用 Excel 的上班族（最适配）
  • 需要批量处理表格数据但不会写 Python 脚本的人
  • 想用 Claude 做数据分析但不想手动敲 SQL
