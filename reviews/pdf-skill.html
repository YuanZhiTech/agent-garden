效率工具文档处理NEW

评测对象

document-skills/pdf — 让 Claude Code 具备 PDF 文件的处理能力。支持文字提取、表格提取、PDF 合并拆分、表单填写。

为什么选这个测

元智报告显示：PDF 批处理大师在 SkillHub 上 120 万+ 下载量，评分 4.8，是所有 Skill 品类里下载量最高的。文档处理是刚需中的刚需。

怎么装

# 从矿脉②克隆（一次克隆，永久使用）
git clone https://github.com/ComposioHQ/awesome-claude-skills.git

# 在 Claude Code 中加载
claude --skill-dir ./awesome-claude-skills/document-skills/pdf

跑起来怎么样

装好后，你让 Claude 做的事：

  • 提取合同关键条款： "帮我把这份 PDF 合同里的甲方乙方、金额、日期提取出来"
  • 合并多个 PDF： "把桌面上这三个 PDF 合并成一个文件"
  • 表格提取： "把这个 PDF 里的表格转成 Excel"
  • 表单填写： "帮我填一下这份 PDF 表单，姓名：张三，日期：今天"

真实效果

  • 文字提取： 准确率高，中文支持好，排版基本保持
  • 表格提取： 简单表格效果好，复杂表格（合并单元格/嵌套）会丢失部分结构
  • PDF 合并拆分： 稳定，大文件（100页+）也流畅
  • 表单填写： 基础表单可以，带校验逻辑的复杂表单可能出问题

踩坑

  1. 扫描版 PDF 不行 — 这个 Skill 处理的是数字生成的 PDF，扫描件（图片版）需要 OCR 工具配合，这个 Skill 不包含 OCR 能力
  2. 中文排版有时乱 — 提取中文 PDF 时，某些特殊字体排版可能会错位
  3. 加密 PDF 要密码 — 有密码保护的 PDF 需要先解密，Claude 会问你要密码

适合谁

  • 经常处理合同、报表、论文的上班族
  • 需要批量提取 PDF 数据的分析师
  • 想用 AI 做文档自动化的开发者
