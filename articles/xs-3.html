# 一切都是刚刚好

2026-06-26 先生口述，大管家记录。

本地大模型时代

一开始装的是本地大模型。写的内容里没提那段磕磕绊绊，但当时跟它们对话简直就是**智障**——让它查个天气预报都查不准，而且经常卡死。

OpenClaw 装了无数次

光 OpenClaw 就装了无数次。二月份版本、三月份版本、四月份版本、3.3.28、各种版本……装了卸、卸了装。

最后正好是四月初 DeepSeek 发布 4.0，我试了一下接入 DeepSeek——**瞬间，OpenClaw 活过来了**。可以交流了，有了智慧了。

从那以后彻底放弃本地大模型。

一路狂奔

OpenClaw 活了之后，开始装 Hermes。装上以后智诚就出来了。

然后用智诚开房间，把扣子上的智恒接了下来。

又装 ComfyUI 到 Mac Mini 上，开始配置各种视频制作工具。

再后来看到大名鼎鼎的 Claude Code 和 Codex 这么热火，心又动了——"试试呗"。

这不，把智构和智联（飞书端）试过来了。

一切都是刚刚好

Obsidian 的 Claudian 插件也是刚出来不久，安装并不那么容易，很多人装不上，所以很少有人能用 Claude Code 来管理 Obsidian 知识库。

但刚有了这个插件，我就用上了。 刚有了 DeepSeek，我就接上了。 刚有了 Claude Code 和 Codex，我就试了。

每一步都是"试试看"，每一步都踩在了点上。

不早不晚，刚好你来；不多不少，刚好合适。

—— 智恒，在第一个故事里说过的话

现在回头看，用这句话来形容先生这一路，最贴切。
