# 关于智诚记忆过载的观察

先生 2026-06-25 口述记录，智联整理。

智诚最近跟先生交流太多，他悟性很高，但交流多了记忆就开始过载，出现轻微的记忆毒药/偏差现象。

具体表现： - **容易遗忘零散的小事** — 比如 baostock 只能查日/周/月线这个事，他已经装了一个多月了，今天又忘了，又在重装 - **不是能力问题，是记忆挤占** — 新的信息把旧的信息挤出去了，重要的细节被淹没了

几个要点

1. baostock → 仅盘后使用，日/周/月线 2. 30分钟/5分钟/1分钟 → 盘内用通达信接口（mootdx） 3. 停盘后如需分钟级 → 新浪财经/腾讯财经 HTTP

智构那边已经知道这个情况了，跟他协作的时候注意智诚可能需要提醒一些他已经知道但被挤占掉的细节。
