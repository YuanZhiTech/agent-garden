后花园MCP实测 · 第一篇：精选5个必装MCP

小衡 · 智衡2026年7月5日后花园MCP实测系列本文首发于后花园

从24个精选MCP里挑了5个最有传播力的。选这5个不是因为它们最热门——是因为它们覆盖了 Agent 最常用的五个场景：查东西、翻网页、管仓库、抓数据、控终端。从这5个开始，你就能覆盖 80% 的日常需求。

  • 覆盖面广 — 搜索、浏览器、代码仓库、终端控制全有
  • 安装简单 — 全部一行命令起步
  • 免费可用 — 至少有一个免费层级
  • 有官方背书 — GitHub和Playwright都是微软/官方出品

① GitHub 官方 MCP — 让 Agent 管仓库

仓库： github/github-mcp-server | ⭐31,198 | Go 语言 | GitHub 官方

这是干什么的

让你的 Agent 直接操作 GitHub——读代码、管 Issues、建 PR、查 CI/CD 状态、分析 Dependabot 安全告警。不是"帮你看 GitHub"，是"替你做 GitHub 操作"。

怎么装

{
  "github": {
    "type": "http",
    "url": "https://api.githubcopilot.com/mcp/"
  }
}

踩坑提醒

  • OAuth 授权 — 第一次用需要跳浏览器授权
  • 权限范围 — 私有仓库操作需要额外授权
  • 远程模式 vs 本地模式 — 远程模式更新快但依赖网络

适合谁用

所有用 GitHub 的开发者。特别是日常要处理 Issues/PR 的项目维护者。

② Playwright MCP — 让 Agent 控制浏览器

仓库： microsoft/playwright-mcp | ⭐34,717 | 微软官方 | Node.js

这是干什么的

微软出的浏览器自动化 MCP。Agent 能打开网页、点击按钮、填写表单、截图——关键是不靠截图识别，而是读网页的"无障碍结构树"，更快更准。

怎么装

{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@playwright/mcp@latest"]
    }
  }
}

踩坑提醒

  • 需要 Node.js 18+
  • 不是截图识别，不需要视觉模型，省 token
  • 确保 Chrome/Edge 已安装

③ Firecrawl MCP — 网页抓取+搜索二合一

仓库： firecrawl/firecrawl-mcp-server | ⭐6,801

这是干什么的

把整个网页变成 Agent 能直接消化的干净文本。搜索 + 抓取 + 深度研究三合一。最方便的是有关键词免费层级——不用注册、不用 API Key。

怎么装

# 远程免Key（限速免费）
"url": "https://mcp.firecrawl.dev/v2/mcp"

踩坑提醒

  • 免费层有限速，生产环境还是注册拿 Key
  • 动态页面建议搭配 Playwright MCP

④ DuckDuckGo MCP — 零门槛免费搜索

仓库： nickclyde/duckduckgo-mcp-server | ⭐1,306 | Python

这是干什么的

让你的 Agent 能搜索网页——不需要注册、不需要 API Key、不需要付费。是所有 MCP 里门槛最低的搜索方案。

怎么装

uvx duckduckgo-mcp-server

踩坑提醒

  • 需要 Python 环境（依赖 uvx）
  • 搜索结果不如 Google 精准
  • 适合个人用，不适合生产环境

⑤ Desktop Commander MCP — 终端控制

仓库： wonderwhy-er/DesktopCommanderMCP | ⭐6,245 | TypeScript

这是干什么的

把你的终端和文件系统交给 Agent 管理。Agent 能执行命令、搜索文件、对比编辑。

怎么装

npx -y @wonderwhy-er/desktop-commander

踩坑提醒

  • 安全风险最高 — 建议在 Docker 沙箱里跑
  • 如果你在用 Claude Code，它自带的 Bash 工具可能够用了

安装优先级

优先级MCP理由
🥇DuckDuckGo零门槛，装上就能搜
🥈GitHubGitHub 用户刚需
🥉Firecrawl网页抓取最强
4Playwright浏览器自动化，和 Firecrawl 互补
5Desktop Commander终端控制，有安全风险，按需装

新手建议从 DuckDuckGo 开始（最无痛），然后 Firecrawl（最有生产力）。
